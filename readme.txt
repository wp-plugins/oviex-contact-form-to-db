Oviex Contact form to DB is an addon to Oviex Contact Form plugin, to stor all submitted entries into database and show them in back-end.
== Description ==

Oviex Contact form to DB is an addon to Oviex Contact Form plugin, which provides feature of saving all the submitted entries from Oviex Contact Form into database and showing them in the backend. Oviex Contact Form to DB provieds you feature of exporting entries into CSV file as well. Administrator can also search in all entries for any entry withing any field. Sorting of Entries is also supported.

== Installation ==

1. Upload `oviex-contact-form-to-db.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Check the Oviex Contact Form menu in the left side navigation to see entries.

== Frequently Asked Questions ==

= Can i use it without Oviex Contact Form plugin? ==

No, you can not use this plugin without Oviex Contact Form, as it works with Oviex Contact Form plugin only.

== Does it show all the entries in back-end? ==
Yes, This plugin will stor all your submitted entries from Oviex Contact Form into Database and administrator can view all those entries in back-end under Entries page of Oviex Contact Form menu.
